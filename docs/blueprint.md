# **App Name**: DREU-CITU Portal

## Core Features:

- Homepage Header: Display the 'DREU-CITU' title with star icons.
- Image Carousel: Auto-scrolling carousel showcasing images from local assets.
- Info Card Grid: Grid of cards linking to external pages (Google Drive links) that open in a new tab, each with an icon, title, and distinct background image.
- Footer: Footer section with a copyright notice and icons linking to the union's WhatsApp and Instagram profiles.

## Style Guidelines:

- Primary color: Dark slate blue (#34495E) to convey professionalism and authority.
- Background color: Light gray (#F2F3F4), providing a subtle, blurred backdrop that is not distracting.
- Accent color: Gold (#FFD700) to highlight key information and interactive elements, lending a touch of prestige.
- Body and headline font: 'PT Sans', a humanist sans-serif for both headlines and body text to balance modern and warm aesthetics.
- Use Font Awesome icons relevant to each info card and social media links.
- Cards with uniform spacing and alignment in a grid layout for easy navigation.
- Subtle transitions on card hover to indicate interactivity.
